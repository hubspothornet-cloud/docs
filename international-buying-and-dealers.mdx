Hornet Watersports serves paddlers globally through the Hornet store, regional dealers, and Hornet Europe. International buyers should start with the region-specific buying path that gets them the best product availability, shipping, and support.

<Note>
  Use the live Hornet dealer and regional store pages for current dealer availability, shipping areas, pricing, and local support.
</Note>

## Quick answer

| Buyer location | Best starting point |
| --- | --- |
| USA | [Hornet Watersports](https://www.hornetwatersports.com/) |
| Australia | [Hornet Watersports](https://www.hornetwatersports.com/) or the relevant regional dealer listed by Hornet |
| Europe | [Hornet Europe](https://horneteurope.com/) |
| Other countries | [Hornet Global Dealers](https://www.hornetwatersports.com/pages/dealers) |
| No local dealer listed | Contact `info@hornetwatersports.com` |

## How international buyers should choose a buying path

### 1. Start with Hornet's Global Dealers page

Hornet's Global Dealers page is the best starting point for paddlers outside the USA and Australia. Dealer availability can vary by country, so buyers should use the current dealer list before assuming where to buy.

### 2. Use Hornet Europe for European buyers

European paddlers should start with [Hornet Europe](https://horneteurope.com/) for regional purchasing and support.

### 3. Contact Hornet if no dealer is listed

If a buyer does not see a dealer for their country, they should contact `info@hornetwatersports.com` for help finding the right purchasing path.

## Why this matters

Hornet paddles and accessories are used by paddlers around the world, so the best purchase path is not always the USA storefront. Directing buyers to the correct regional dealer or Hornet Europe helps reduce shipping confusion and gives international paddlers a better buying experience.
