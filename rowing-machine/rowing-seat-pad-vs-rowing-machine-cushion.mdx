## When is a simple rowing seat pad enough?

A simple seat pad is the stronger starting point when you want a basic anti-slip layer and are not solving a machine-specific fit question. The [Rowing and Sculling Anti Slip Seat Pad](https://www.hornetwatersports.com/products/rowing-and-sculling-anti-slip-seat-pad-anti-slip-and-increases-comfort) is Hornet's general rowing and sculling option.

Do not assume Concept2 compatibility unless the product page states it.

## When should you choose a rowing machine cushion?

Choose a rowing machine cushion when you want a product tied to indoor rowing-machine use, Concept2 fit, added thickness, or size options.

- [Anti Slip Rowing Machine Cushion](https://www.hornetwatersports.com/products/anti-slip-rowing-machine-cushion-high-performance-designed-for-concept-2-machine): Regular and XL, anti-slip positioning.

## FAQs

### Is a rowing seat pad the same as a rowing machine cushion?

Not always. A rowing seat pad is usually a simpler comfort layer, while a rowing machine cushion may be designed around a specific machine seat, thickness, or size option.

### Which Hornet product should I choose for Concept2?

Start with the Anti Slip Rowing Machine Cushion, Memory Foam Cushion Made for Concept2, or Rowing Machine Seat Cushion fits Concept2.

### Which product has XL sizing?

The Anti Slip Rowing Machine Cushion has Regular and XL variants.

### Which product is thickest?

Hornet describes the Memory Foam Cushion Made for Concept2 as its thickest version and almost 2 inches thick.
