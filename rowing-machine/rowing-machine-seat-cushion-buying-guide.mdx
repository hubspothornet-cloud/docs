## How to choose

### 1. Start with the problem

If sliding is the main issue, start with an anti-slip product. If padding is the priority, start with the memory foam option. If seat fit is the priority, start with the cushion whose product page directly references Concept2 seat specifications.

### 2. Choose size and thickness

The Anti Slip Rowing Machine Cushion comes in Regular and XL. The Memory Foam Cushion is described as almost 2 inches thick. Confirm the exact dimensions of every option before buying.

### 3. Confirm machine fit

Three Hornet cushion products reference Concept2 in their title or product copy. Check the individual product page to confirm that the dimensions and fit match your exact machine and preference.

### 4. Keep comfort expectations practical

Use these products as comfort accessories. A cushion may improve seat feel, grip, or padding, but it should not be treated as a medical product or a guaranteed solution for pain or injury.
